# Ventas

El módulo de **ventas** proporciona las herramientas necesarias para gestionar de manera eficiente todo el ciclo de ventas de la empresa. Desde la creación de **propuestas** y **proformas**, hasta la emisión de **facturas**, **notas de crédito** y el registro de **cobros**, este módulo centraliza y automatiza las operaciones comerciales, asegurando la precisión en los documentos financieros y facilitando el seguimiento detallado de cada transacción. Además, facilita la integración contable y la generación de análisis estratégicos que optimizan la toma de decisiones comerciales. El ciclo inicia con la creación de propuestas para clientes, seguida por la generación de proformas que detallan los términos acordados. Finalmente, se emite la factura que formaliza la transacción. {{className:'lead text-justify'}}

## Funcionalidades

Desde el módulo de ventas podemos:

1. Crear, editar y eliminar propuestas.
2. Crear, editar y eliminar proformas.
3. Administrar facturas.
4. Crear, editar, eliminar notas de crédito.
5. Registrar, asignar pagos y eliminar cobros.

## Proformas

Una **proforma** es un documento comercial provisional que detalla los productos o servicios ofrecidos junto con sus costos estimados. Se emite antes de la confirmación final de una transacción, proporcionando a los clientes una estimación clara de lo que se espera que paguen. A diferencia de una factura, una proforma no es legalmente vinculante y puede ser modificada antes de convertirse en una transacción finalizada.
Una proforma puede ser creada de dos maneras:{{className:'text-justify'}}

1. Convirtiendo una propuesta a proforma.
2. Creando una nueva proforma.

Desde la sección de proformas podemos:

- Crear una nueva proforma.

## Facturas

Una **factura** es un documento comercial que detalla la venta de bienes o servicios, indicando el precio, impuestos aplicables, y otras condiciones de la transacción. Sirve como prueba de la operación y facilita el registro contable tanto para el vendedor como para el comprador. Desde la sección de facturas podemos:{{className:'text-justify'}}

- Ver detalles e imprimir factura.
- Listado de cobros de una factura.
- Listado de créditos de una factura.
- Crear tareas a una factura.

## Notas de crédito

Una **nota de crédito** es un documento emitido por un vendedor a un comprador para corregir errores en una factura previamente emitida. Se utiliza para ajustar el monto adeudado por el comprador, ya sea para devolver dinero, otorgar crédito para futuras compras, o corregir errores como cantidades incorrectas o precios erróneos. Es una herramienta importante para mantener registros precisos y resolver disputas comerciales de manera justa y eficiente. Desde esta sección podemos:{{className:'text-justify'}}

1. Registrar una nueva nota de crédito.
