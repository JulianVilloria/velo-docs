# Casos

Un **caso** en el ámbito legal es una disputa o controversia entre partes que se resuelve a través del sistema judicial, basado en hechos y argumentos legales específicos.
Cada caso se clasifica según su tipo, que puede ser administrativo, judicial, laboral, entre otros. Cada tipo de caso conlleva **procesos** específicos asociados y cada proceso tiene **procedimientos** que se deben llevar a cabo.{{ className: 'lead text-justify' }}

Un procedimiento puede entenderse como una guía o plantilla detallada que describe paso a paso cómo realizar una tarea específica de manera consistente y eficiente. Por ejemplo, si tengo un caso laboral, el proceso comúnmente conlleva ciertos procedimientos como la presentación de la demanda, notificación al empleador, entrevistas, etc.{{ className: 'lead text-justify' }}

Al asignar tareas a un caso, si éste pertenece a un tipo que tiene un proceso definido con procedimientos específicos, podemos convertir directamente los procedimientos asociados a dicho proceso a tareas utilizando el botón azul correspondiente. Además, tenemos la opción de crear tareas adicionales que sean específicas y relevantes para ese caso en particular. {{ className: 'lead text-justify' }}

## Funcionalidades

Desde el módulo de casos podemos:

1. Ver estadísticas como:

- La cantidad de casos **no iniciados**.
- La cantidad de casos **en desarrollo**.
- La cantidad de casos **en espera**.
- La cantidad de casos **finalizados**.
- La cantidad de casos **cancelados**.

Estas estadísticas corresponden a la cantidad de casos en cada uno de los estados posibles.

2. Agregar un nuevo caso incluyendo datos como costo total, horas estimadas, descripción, demandado y demandante, estado, tipo de servicio, tipo de facturación, persona responsable, miembros del caso, fecha de inicio y fecha de entrega. {{ className: 'text-justify' }}
3. Desde el listado de casos podemos:
   - Acceder a una vista rápida para ver detalles del caso.
   - Editar un caso.
   - Agregar notas a un caso.
   - Eliminar un caso.
   - Registrar un gasto a un caso.

### Registrar proformas

Una proforma es un documento preliminar que proporciona detalles anticipados sobre una transacción comercial o financiera. Es utilizado para informar al cliente sobre los términos y condiciones de una venta o acuerdo antes de que se realice la transacción definitiva.{{ className: 'text-justify' }}

<Image src="/images/casos-9.png" alt="Clientes" width={700} height={500} />

### Ver las facturas del caso

Una factura es un documento comercial que detalla la venta de bienes o servicios entre un vendedor y un comprador. Es un comprobante legal y financiero que establece las obligaciones financieras entre ambas partes. Desde esta vista podremos acceder a un listado de todas las facturas del caso.{{ className: 'text-justify' }}

<Image src="/images/casos-10.png" alt="Clientes" width={700} height={500} />
