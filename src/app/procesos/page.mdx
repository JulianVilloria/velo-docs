# Procesos

Un proceso se refiere a una serie estructurada y sistemática de acciones o pasos que se llevan a cabo de manera secuencial o simultánea para alcanzar un objetivo específico. Describen la serie de pasos que deben seguirse para tratar un caso en particular.
Cada caso del sistema se clasifica según su tipo, que puede ser administrativo, judicial, laboral, entre otros. Cada tipo de caso conlleva procesos específicos asociados y cada proceso tiene procedimientos que se deben llevar a cabo.
Un procedimiento puede entenderse como una guía o plantilla detallada que describe paso a paso cómo realizar una tarea específica de manera consistente y eficiente.
Por ejemplo, si tengo un caso laboral, el proceso comúnmente conlleva ciertos procedimientos como la presentación de la demanda, notificación al empleador, entrevistas, etc.{{className:'text-justify'}}

## Funcionalidades

Desde el módulo de procesos podemos:

1. Crear un nuevo proceso con sus correspondientes datos.
2. Editar un proceso.
3. Eliminar un proceso.
4. Agregar procedimientos al proceso.
5. Desde el listado de procedimientos podemos:

- Agregar un nuevo procedimiento.
- Editar un procedimiento.
- Eliminar el procedimiento.
- Cambiar el orden de los procedimientos.
